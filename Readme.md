**Cardio-Readmit Risk Advisor** — це інтерактивний, автономний односторінковий вебдодаток (Single-Page Application), розроблений для кардіохірургів, анестезіологів та лікарів інтенсивної терапії. Інструмент допомагає оцінити індивідуальний ризик незапланованої повторної госпіталізації пацієнта протягом 30 діб після кардіохірургічного втручання та автоматично генерує персоналізований план виписки.

## ✨ Основні можливості

*   ⚡ **Реактивна модель обчислень**: Миттєвий перерахунок ризику при зміні будь-якого клінічного параметра без перезавантаження сторінки.
*   🔄 **Динамічні модифікатори**: На відміну від статичних шкал, модель враховує динаміку відновлення (наприклад, щоденну кількість кроків перед випискою та стабільність ран).
*   🖨️ **Експорт для пацієнта (Print-Ready)**: Спеціальні CSS-стилі для друку генерують лаконічну, чорно-білу пам'ятку формату А4 для пацієнта з графіком прийому ліків, червоними прапорцями та правилами догляду.
*   🔒 **Абсолютна конфіденційність (Zero-Backend)**: Додаток працює на 100% локально в браузері. Медичні дані пацієнтів не відправляються на сервери, що повністю відповідає вимогам GDPR та HIPAA.
*   📱 **Адаптивний дизайн**: Інтерфейс оптимізовано як для десктопних моніторів у ординаторській, так і для планшетів/смартфонів.

## 🛠 Технічний стек

Проєкт реалізовано у філософії *Zero Dependencies* (для розгортання) в одному файлі `index.html`:
*   **Розмітка та Логіка**: Чисті HTML5 та Vanilla JavaScript (ES6+).
*   **Стилізація**: Tailwind CSS (підключається через CDN для миттєвої роботи).
*   **Іконки**: Інтегровані in-line SVG.
